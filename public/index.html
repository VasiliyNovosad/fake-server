API Endpoints:

GET /api/chats - returns JSON list of chats

GET /api/chats/:chat_id/messages - returns JSON list of chat's messages

GET /api/chats/:chat_id/messages/:id - returns JSON message with id=:id

GET /api/chats/:chat_id/messages/:id/attachment - downloads pdf document for message with id=:id

GET /pdfs/${:message_id}FullNote.pdf - downloads pdf document
